GAN-DexGrasp \(=\) Iterative Self-Improving Framework for Dexterous Grasping

Abstract

Robust dexterous grasping requires models to generalize beyond the object geometries observed during training. However, existing learning-based methods often fail on rare or geometrically challenging objects that expose previously unseen failure modes. We propose \textbf{GAN-DexGrasp}, an iterative self-improving framework that continuously discovers and learns from such failures through adversarial object generation. Our method learns a rollout-based performance predictor and uses it to synthesize challenging yet realistic objects via diffusion-regularized latent optimization. These objects are then used to collect additional grasp data and retrain the grasping model, progressively improving robustness and generalization. Experiments demonstrate consistent gains over random object generation, dataset sampling, latent-space deformation, and direct 3D-space deformation. The results suggest that adversarial object generation is an effective and scalable strategy for improving dexterous grasping.
